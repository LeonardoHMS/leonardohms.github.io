Project

OP Calculator

OP-Calculator is a graphical interface program designed to simplify certain work processes with the SAP system. It calculates the hours worked for time tracking, marking worked hours in minutes for production, and also calculates the weight of scraps. The program also integrates with the SAP GUI Scripting API, helping to analyze production orders by retrieving all necessary information through transactions, extracting spreadsheets, and filtering only the orders that are set to close technically (ENTE).

Project

Calculator for TRUMPF

Calculator for TRUMPF is a Node.js-based code designed to extract production time data provided by the TRUMPF cutting machine. The data is extracted from the HTML file generated by the machine's own system, performing calculations to analyze the actual production time. The software's name is still temporary, and the entire code is still under development.
